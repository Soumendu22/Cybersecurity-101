# Linux Permissions

- Permission Model
    
    - user, group, others
    
    - read, write, execute
    
- Permission Formats
    
    - symbolic mode
    
    - octal mode
    
- Ownership
    
    - chown
    
    - chgrp
    
- Special Bits
    
    - SUID
    
    - SGID
    
    - Sticky bit
    
- Advanced Control
    
    - umask
    
    - ACLs
    
- Security Checks
    
    - world-writable files
    
    - SUID enumeration
