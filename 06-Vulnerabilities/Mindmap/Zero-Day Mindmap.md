# Zero-Day

- Zero-Day Characteristics
    
    - no immediate patch
    
    - active exploitation risk
    
- Response Steps
    
    - identify affected assets
    
    - deploy compensating controls
    
    - increase detection coverage
    
- Controls
    
    - segmentation and isolation
    
    - WAF/IPS mitigations
    
    - emergency patching workflow
    
- Governance
    
    - communication cadence
    
    - incident lessons learned
