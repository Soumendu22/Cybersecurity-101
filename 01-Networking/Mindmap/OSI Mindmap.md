# OSI Model

- Application Layer (Layer 7)
    
    - User interaction
        
    - Protocols
        
        - HTTP
            
        - HTTPS
            
        - FTP
            
        - SMTP
            
        - DNS
            
    - Attacks
        
        - XSS
            
        - SQL Injection
            
        - Authentication bypass
            
- Presentation Layer (Layer 6)
    
    - Data formatting
        
    - Encryption
        
    - Compression
        
    - Technologies
        
        - SSL
            
        - TLS
            
        - JPEG
            
        - MPEG
            
- Session Layer (Layer 5)
    
    - Session establishment
        
    - Session management
        
    - Session termination
        
    - Security issues
        
        - Session hijacking
            
        - Session fixation
            
- Transport Layer (Layer 4)
    
    - End to end communication
        
    - Protocols
        
        - TCP
            
        - UDP
            
    - Ports
        
        - 80 HTTP
            
        - 443 HTTPS
            
        - 22 SSH
            
        - 53 DNS
            
    - Attacks
        
        - SYN Flood
            
        - DoS
            
- Network Layer (Layer 3)
    
    - Routing packets
        
    - Logical addressing
        
    - Protocols
        
        - IP
            
        - ICMP
            
        - IPSec
            
    - Attacks
        
        - IP Spoofing
            
        - Routing attacks
            
- Data Link Layer (Layer 2)
    
    - MAC addressing
        
    - Frame transmission
        
    - Protocols
        
        - Ethernet
            
        - ARP
            
        - PPP
            
    - Attacks
        
        - ARP Poisoning
            
        - MAC Spoofing
            
- Physical Layer (Layer 1)
    
    - Transmission of bits
        
    - Hardware components
        
        - Ethernet cables
            
        - Fiber optics
            
        - Network interface cards
            
    - Attacks
        
        - Cable tapping
            
        - Hardware tampering
