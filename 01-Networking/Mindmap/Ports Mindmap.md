# Network Ports

- Port Concept
    
    - Communication endpoint
        
    - Works with IP address
        
    - Identifies services
        
- Port Ranges
    
    - Well Known Ports
        
        - 0 – 1023
            
    - Registered Ports
        
        - 1024 – 49151
            
    - Dynamic Ports
        
        - 49152 – 65535
            
- Protocol Types
    
    - TCP
        
        - Reliable
            
        - Connection oriented
            
    - UDP
        
        - Fast
            
        - Connectionless
            
- Important Ports
    
    - 21 FTP
        
    - 22 SSH
        
    - 23 Telnet
        
    - 25 SMTP
        
    - 53 DNS
        
    - 80 HTTP
        
    - 110 POP3
        
    - 139 NetBIOS
        
    - 143 IMAP
        
    - 443 HTTPS
        
    - 445 SMB
        
    - 3306 MySQL
        
    - 3389 RDP
        
- Security Attacks
    
    - SSH brute force
        
    - FTP credential attacks
        
    - Web exploitation
        
    - SMB exploits
        
    - RDP brute force
        
- Tools Using Ports
    
    - Nmap
        
    - Netcat
        
    - Wireshark
        
    - Metasploit
        
- Firewall
    
    - Allow ports
        
    - Block ports
        
    - Monitor traffic
