# TCP/IP Model

- Application Layer
    
    - Provides services to applications
        
    - Protocols
        
        - HTTP
            
        - HTTPS
            
        - DNS
            
        - FTP
            
        - SMTP
            
        - POP3
            
        - IMAP
            
        - SSH
            
        - SNMP
            
- Transport Layer
    
    - End to end communication
        
    - Protocols
        
        - TCP
            
            - Reliable
                
            - Connection oriented
                
            - 3 way handshake
                
        - UDP
            
            - Fast
                
            - Connectionless
                
            - Used in streaming and gaming
                
- Internet Layer
    
    - Routing packets
        
    - Logical addressing
        
    - Protocols
        
        - IP
            
        - ICMP
            
        - ARP
            
        - IPSec
            
- Network Access Layer
    
    - Physical network communication
        
    - Technologies
        
        - Ethernet
            
        - WiFi
            
        - PPP
            
- Data Flow
    
    - Application → Request
        
    - Transport → Segmentation
        
    - Internet → Routing
        
    - Network Access → Transmission
        
- Security Attacks
    
    - SYN Flood
        
    - IP Spoofing
        
    - ARP Poisoning
        
    - Session Hijacking
