# Subnetting

- Subnetting Concept
    
    - Divide large network
        
    - Improve security
        
    - Improve performance
        
- IP Address
    
    - Network portion
        
    - Host portion
        
- Subnet Mask
    
    - Defines network bits
        
    - Example
        
        - 255.255.255.0
            
- CIDR Notation
    
    - /24
        
    - /16
        
    - /8
        
- Private IP Ranges
    
    - 10.0.0.0
        
    - 172.16.0.0
        
    - 192.168.0.0
        
- Address Types
    
    - Network address
        
    - Broadcast address
        
    - Host addresses
        
- Example Subnet
    
    - 192.168.1.0/24
        
    - Host range
        
        - 192.168.1.1
            
        - 192.168.1.254
            
- Cybersecurity Use
    
    - Network segmentation
        
    - Attack containment
        
    - Monitoring
