# Cross-Site Request Forgery (CSRF)

- Attack Model
    
    - victim authenticated
    
    - browser auto-sends cookies
    
    - attacker triggers unwanted action
    
- Targets
    
    - profile changes
    
    - password reset actions
    
    - payment/admin operations
    
- Defenses
    
    - anti-CSRF tokens
    
    - SameSite cookies
    
    - origin/referer validation
    
    - step-up authentication
