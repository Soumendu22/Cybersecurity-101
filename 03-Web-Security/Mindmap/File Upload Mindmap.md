# File Upload Security

- Upload Risks
    
    - webshell execution
    
    - MIME/extension bypass
    
    - path traversal filenames
    
- Core Defenses
    
    - strict allowlists
    
    - magic-byte validation
    
    - random server filenames
    
    - non-executable storage
    
- Additional Controls
    
    - malware scanning
    
    - size/rate limits
    
    - access control checks
