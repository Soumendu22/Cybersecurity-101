# IDOR

- Object-Level Access
    
    - direct ID references
    
    - ownership checks required
    
- Attack Paths
    
    - modify numeric IDs
    
    - enumerate UUID/resource keys
    
- Impact
    
    - unauthorized data access
    
    - unauthorized updates/deletions
    
- Prevention
    
    - server-side authorization per object
    
    - policy enforcement middleware
    
    - indirect reference mapping
